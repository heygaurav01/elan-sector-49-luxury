# Elan Sector 49 - Luxury Apartments

This project is a React application for Elan Sector 49 Luxury Apartments in Gurgaon.

## Project info

Built with:
- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

## Building for Production

To build the project for production:
```sh
npm run build
```
